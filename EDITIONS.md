# GeoLens open source

GeoLens is Apache-2.0 open-source software for self-hosting a spatial data catalog and map builder on infrastructure you control.

GeoLens has one public distribution in this repository: the Apache-2.0 self-hosted application.

This repository contains the self-hosted GeoLens application for one organization's spatial data catalog and map builder. It includes the backend, frontend, CLI, generated SDKs, Docker Compose setup, examples, and release workflows needed to run GeoLens yourself.
