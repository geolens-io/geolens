# SAML SSO

SAML SSO requires the GeoLens Enterprise overlay. Community edition does not
include SAML routes, SAML provider setup, or SAML operator runbooks.

The public GeoLens repository owns the extension points and community-safe
edition gates only. Detailed SAML installation, IdP configuration, hardening,
troubleshooting, and lifecycle guidance live with the private
`geolens-enterprise` source.

Community deployments should use the built-in local auth and OAuth/OIDC
provider configuration documented in the public admin and configuration guides.
