# Enterprise Reactivation

Enterprise reactivation is an operator runbook for deployments that previously
used the GeoLens Enterprise overlay. It depends on the private overlay package,
enterprise migrations, and any preserved SAML state, so the detailed procedure
lives with the private `geolens-enterprise` source.

The public GeoLens repository keeps the community runtime, extension contracts,
and edition gates. Community deployments do not need an enterprise reactivation
procedure.
