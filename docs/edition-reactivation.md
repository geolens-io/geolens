# Edition Reactivation Runbook

Reactivating the GeoLens **Enterprise** overlay (re-enabling enterprise-only features after a previous deactivation) is an operator-facing flow shipped with the private `geolens-enterprise` source. Reactivation preserves audit-log history, restores SAML provider configuration, and re-enables enterprise UI surfaces (branding, advanced sharing, audit log export).

For step-by-step reactivation procedures (including overlay package installation, secret restoration, and round-trip data preservation verification), see the operator guide on the documentation site:

- **Edition reactivation runbook:** [docs.getgeolens.com/guides/operations/edition-reactivation/](https://docs.getgeolens.com/guides/operations/edition-reactivation/)

For information on obtaining or licensing the GeoLens Enterprise overlay, see the README **Enterprise and Security** section or contact `enterprise@getgeolens.com`.
