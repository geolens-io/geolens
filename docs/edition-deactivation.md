# Enterprise Deactivation

Enterprise deactivation is an operator runbook for the GeoLens Enterprise
overlay. It includes SAML data-fate decisions, rollback planning, and
commercial deployment steps, so the detailed procedure lives with the private
`geolens-enterprise` source.

The public GeoLens repository keeps the community runtime, extension contracts,
and edition gates. Community deployments do not have an enterprise overlay to
deactivate.
