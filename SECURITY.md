# Security Policy

## Reporting a Vulnerability

Do **not** report security vulnerabilities through public GitHub issues or discussions.

Please email **security@getgeolens.io** with a description of the vulnerability, steps to reproduce, and any relevant details. We will acknowledge receipt within 48 hours and provide an estimated timeline for a fix.

For more details, see the [Reporting Security Issues](https://github.com/geolens-io/geolens/blob/main/.github/CONTRIBUTING.md#reporting-security-issues) section of our contributing guidelines.
