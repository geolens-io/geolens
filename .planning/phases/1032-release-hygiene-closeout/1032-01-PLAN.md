# Phase 1032 Plan: Release Hygiene Closeout

## Goal

Close the post-v1006 release hygiene milestone by verifying dependency alerts, generated artifacts, local stack health, smoke coverage, and temporary UAT data cleanup without changing product scope.

## Requirements

- REL-01: Verify open Dependabot alerts against the actual lockfile and local dependency scanners.
- REL-02: Run backend security, lint, format, and full test coverage gates.
- REL-03: Run frontend i18n, lint, typecheck, and coverage gates.
- REL-04: Reconcile OpenAPI and generated SDK drift from v1006.
- REL-05: Bring the compose stack to a healthy state and fix any release-blocking healthcheck drift.
- REL-06: Run Playwright smoke coverage across core, builder, and fixture flows.
- REL-07: Use Playwright MCP for a live browser sanity check and current-page console review.
- REL-08: Remove temporary v1006/UAT and smoke datasets that pollute authenticated search results.
- REL-09: Archive evidence and known caveats for milestone handoff.

## Approach

1. Query GitHub Dependabot alerts and compare them against `backend/pyproject.toml`, `backend/uv.lock`, `pip-audit`, and `bandit`.
2. Run backend and frontend CI-parity gates from the current CI workflow and package scripts.
3. Regenerate OpenAPI/SDK artifacts if drift is real and inspect the resulting diff.
4. Restart the compose stack, fix healthcheck issues that block `docker compose up --wait`, and rerun stack health.
5. Run `npm run e2e:smoke`; fix stale fixture assumptions only when the failure is deterministic and release-hygiene scoped.
6. Use Playwright MCP to verify the live search page renders and has no current-page warnings/errors after temp-data cleanup.
7. Update planning archives, state, and milestone summaries.

## Success Criteria

- Dependency scanners are green and Dependabot alerts are either resolved locally or documented as stale pending GitHub rescan.
- Release gates pass or any environment-only failure is explicitly documented.
- OpenAPI and SDK generated artifacts are aligned with current backend routes.
- Compose `--wait` succeeds.
- Playwright smoke and MCP browser sanity pass.
- v1007 planning artifacts provide a complete closeout trail.
