The translation key `common:pageTitle.notFound` does not exist today — add it to en/de/es/fr alongside the existing `pageTitle.login`, `pageTitle.register`, etc.

Purpose: Close ROUTE-02 — tab UX consistency.

    export function useDocumentTitle(title: string) {
      useEffect(() => {

      }, [title]);
    }
