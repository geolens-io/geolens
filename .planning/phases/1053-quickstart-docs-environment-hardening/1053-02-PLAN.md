  ## 4. Seed sample data

  After login, your catalog is empty. The fastest way to populate it is one of
  the two bundled API seeders. Both write into a running GeoLens instance via
  the upload/service-import API — no separate stack, no separate compose file.

  | Seeder | Source | Datasets | When to use |
  |---|---|---|---|
  | `seed-natural-earth.py` | Natural Earth 1:10m vector | ~109 cultural + physical | Offline-friendly, deterministic, fast to re-seed |
  | `seed-ago-data.py` | Public ArcGIS Online org (NJ Highlands default) | ~78 layers from public services | Live data, demonstrates the AGO connector |

  Both seeders need Python 3.10+ and `httpx` — see [Prerequisites](#prerequisites)
  above. The seeders run on your host machine and talk to the running GeoLens
  at `http://localhost:8080`.

  ### Seed Natural Earth (offline)

  ```bash
  pip install httpx
  python scripts/seed-natural-earth.py --username admin --password admin
  ```

  This downloads ~109 Natural Earth shapefile zips from `naciscdn.org`, ingests
  each into GeoLens, and emits a one-line progress entry per dataset. Total
  runtime is roughly 3-5 minutes on a typical broadband connection (most of
  it is the CDN download).

  ### Seed live ArcGIS Online data

  `seed-ago-data.py` requires a pre-created API key. See the next section for
  the 3-click API-key recipe; once you have a key, run:

  ```bash
  pip install httpx
  python scripts/seed-ago-data.py --api-key <your-key>
  ```

  By default this discovers public layers from the NJ Highlands AGO org. Pass
  `--org-url <url>` to point at a different organization, or `--dry-run` to
  list discoverable layers without ingesting.

  See [Create your first API key](#create-your-first-api-key) below for the
  API-key bootstrap workflow.
  ```

  conflict-events stories.

  compose file as production. The bake-time demo is the better fit when you
