# Phase 999.4 — Security

> Per-phase security contract: threat register, accepted risks, and audit trail.

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| `file_path` → ogr2ogr | Untrusted file content processed by GDAL | Vector file bytes (user-uploaded) |
| `target_table` → SQL | Table name used in SQL statements | Caller-generated identifier (not user-supplied) |
| test fixtures → test DB | Test data written to isolated test database | Synthetic GeoJSON / CSV |
