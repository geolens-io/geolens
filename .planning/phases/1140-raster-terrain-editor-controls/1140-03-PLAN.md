<objective>
Add EDITOR-DEM-05: a HYPSOMETRIC TINT section to `DEMEditorScene` (hillshade mode only) and the companion native MapLibre `color-relief` layer that tints the DEM by elevation using a curated preset ramp.

Purpose: MapLibre GL JS 5.24 ships a native `color-relief` layer (RESEARCH Finding 3) — zero new packages, zero backend change. The control writes builder-private `_hypso-*` paint keys; a new `color-relief-sync.ts` module adds a `color-relief` companion layer on the SAME raster-dem source the hillshade layer already owns, inserted below the hillshade layer so shading renders on top. Because `color-relief-color` is a `ColorRampProperty`, the layer is recreated (remove+add) when the ramp changes rather than via `setPaintProperty` (RESEARCH Pitfall 1).

Output: new `color-relief-sync.ts`; one wiring call in `syncLayersToMap`; the HYPSOMETRIC TINT section in `DEMEditorScene`; unit tests; en/de/es/fr i18n keys.

Sequenced after Plan 02 (Wave 2) because it co-edits `DEMEditorScene.tsx` and `map-sync.ts` with the contour plan.
</objective>

