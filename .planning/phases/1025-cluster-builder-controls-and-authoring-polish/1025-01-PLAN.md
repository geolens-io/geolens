# Phase 1025 Plan: Cluster Builder Controls And Authoring Polish

**Phase:** 1025 — cluster-builder-controls-and-authoring-polish
**Milestone:** v1005 Builder Point Cluster Foundation
**Status:** Complete
**Requirements:** CLUS-06

## Goal

Make Cluster usable in the layer inspector with focused controls for native MapLibre cluster source and label styling, without adding schema fields or UI primitives.

## Implementation

1. Extend point render mode authoring.
   - Add `Cluster` to the point render selector when the layer is eligible or already stores cluster intent.
   - Route Cluster switching through the existing renderAs patch path.

2. Add cluster controls.
   - Radius and max zoom write `style_config.builder.clusterRadius` and `clusterMaxZoom`.
   - Cluster color, count text color, and count text size write existing builder JSON.
   - Keep normal point appearance and layer opacity controls available for unclustered points.

3. Keep live map updates coherent.
   - Cluster color/count label changes sync through the adapter.
   - Cluster source option changes trigger a clustered source rebuild through normal map-sync.

4. Add i18n and focused tests.
   - Localize new builder strings for en/es/fr/de.
   - Cover controls rendering and style-config writes in `LayerStyleEditor` tests.
