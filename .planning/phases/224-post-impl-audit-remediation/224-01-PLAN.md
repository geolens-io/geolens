1. Add `import asyncio` at the top of `backend/app/storage/local.py`.

2. For `put(self, key, data)`: Read all bytes from BinaryIO BEFORE passing to to_thread (BinaryIO is not thread-safe). Pattern:
   ```python
   async def put(self, key: str, data: BinaryIO | bytes) -> str:
       dest = self.base_dir / key
       if not isinstance(data, bytes):
           data = data.read()  # read in async context before thread handoff

       def _put():
           dest.parent.mkdir(parents=True, exist_ok=True)
           dest.write_bytes(data)
           return str(dest)

       return await asyncio.to_thread(_put)
   ```

3. For `get(self, key)`:
   ```python
   async def get(self, key: str) -> bytes:
       path = self.base_dir / key
       return await asyncio.to_thread(path.read_bytes)
   ```

4. For `delete(self, key)`:
   ```python
   async def delete(self, key: str) -> None:
       path = self.base_dir / key
       await asyncio.to_thread(path.unlink, True)  # missing_ok=True
   ```

5. For `exists(self, key)`:
   ```python
   async def exists(self, key: str) -> bool:
       path = self.base_dir / key
       return await asyncio.to_thread(path.exists)
   ```

6. For `copy(self, src_key, dst_key)`:
   ```python
   async def copy(self, src_key: str, dst_key: str) -> str:
       src = self.base_dir / src_key
       dst = self.base_dir / dst_key

       def _copy():
           dst.parent.mkdir(parents=True, exist_ok=True)
           shutil.copy2(str(src), str(dst))
           return str(dst)

       return await asyncio.to_thread(_copy)
   ```

7. For `list_keys(self, prefix)` (if it uses `Path.rglob` or `iterdir`):
   ```python
   async def list_keys(self, prefix: str = "") -> list[str]:
       base = self.base_dir / prefix if prefix else self.base_dir

       def _list():
           if not base.exists():
               return []
           return [str(p.relative_to(self.base_dir)) for p in base.rglob("*") if p.is_file()]

       return await asyncio.to_thread(_list)
   ```

8. For `health_check(self)`:
   ```python
   async def health_check(self) -> bool:
       return await asyncio.to_thread(self.base_dir.exists)
   ```
