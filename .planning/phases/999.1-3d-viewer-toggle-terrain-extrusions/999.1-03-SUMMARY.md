## Summary

Added 3D terrain toggle to ViewerMap using MapLibre's built-in TerrainControl (D-01: zero custom UI). Pre-seeds a `raster-dem` terrain source on map load when DEM layers are detected, toggles terrain on/off with pitch animation, and re-syncs terrain through basemap changes.

## Key Changes

- `TerrainControl` from `@vis.gl/react-maplibre` rendered conditionally when `terrainReady` is true
- `seedTerrainSource` helper pre-seeds `raster-dem` source with terrain tile URL
- `terrain` event listener drives `easeTo` pitch animation (45° on, 0°/0° off)
- `style.load` handler re-seeds terrain source and re-enables terrain after basemap swap
- Fixed `dataset_record_type` check to match `raster_dataset` (not `raster`)
- Fixed shared map `tile_url` for raster layers to use raster-proxy path
