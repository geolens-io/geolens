### Height Column Dropdown

Location: inside the polygon (`fill`) section of `LayerStyleEditor.tsx`, below the existing fill/stroke controls.

Render condition: only when `layer.dataset_geometry_type === 'Polygon'` or `'MultiPolygon'` AND `layer.dataset_column_info` contains at least one numeric column.

Component structure:
```
<div className="flex items-center justify-between gap-2">
  <span className="text-xs text-muted-foreground">Height column</span>
  <Select value={heightCol ?? ''} onValueChange={...}>
    <SelectTrigger className="h-8 text-xs w-36">
      <SelectValue placeholder="None" />
    </SelectTrigger>
    <SelectContent>
      <SelectItem value="">None</SelectItem>
      {numericColumns.map(col => (
        <SelectItem key={col.name} value={col.name}>{col.name}</SelectItem>
      ))}
    </SelectContent>
  </Select>
</div>
```

Numeric column filter: include columns where `type` matches any of: `integer`, `bigint`, `smallint`, `numeric`, `real`, `double precision`, `float4`, `float8`, `int2`, `int4`, `int8`. Exclude text, boolean, geometry, date, uuid types.

### Height Column Removed Warning

Render condition: `paint['_height_column']` is set AND `schemaDiff?.columns_removed` contains a column matching that name (from existing re-upload schema diff infrastructure, per D-12).

Visual: inline `<div>` with `--warning` background, `--warning-foreground` text, 8px padding, 4px border-radius, 12px text. Use Lucide `AlertTriangle` icon (16px) inline-left of the message text.

Copy: "Height column "{columnName}" was removed during re-upload. Select a new column or clear this setting."

### TerrainControl

MapLibre's built-in `TerrainControl`. Added imperatively via `map.addControl(new TerrainControl({ source: 'terrain-dem', exaggeration: 1.5 }), 'top-right')` (per D-01). No custom styling applied — use MapLibre's default control appearance, which matches other MapLibre controls already on the map (zoom, navigation).

Position: top-right of the map canvas, below any existing controls, above the attribution control.

## Interaction States

### 3D Toggle (TerrainControl)

| State | Behavior |
|-------|----------|
| Inactive (default) | Map renders in 2D, standard pitch 0, rotation disabled |
| Activated | Pitch animates smoothly to 45° over 300ms (see Animation section); pitch/rotate navigation enabled; fill-extrusion layers appear if height column is configured; terrain applied if DEM layers present |
| Activated, no DEM | Flat map with pitch/rotate enabled. No visual error. No toast. No warning. (Per D-07) |
| Activated, no height-configured layers | Same as above — pitch/rotate on flat map. No error. (Per D-07) |
| Session reset | TerrainControl state resets to 2D on page reload. No persistence. (Per D-09) |

### Height Column Dropdown

| State | Behavior |
|-------|----------|
| Unset (default) | Shows "None" placeholder. No extrusion layer rendered. |
| Column selected | `paint._height_column` updated via `onPaintChange`. If map is in 3D mode, extrusion companion layer appears. |
| Bound column removed after re-upload | Warning message shown inline (see component inventory). Extrusion layer silently skipped (height column absent from source). |

### Fill-Extrusion Layer

| State | Behavior |
|-------|----------|
| 2D mode, height column set | `fill-extrusion` layer NOT added. Standard `fill` layer shown. |
| 3D mode, no height column | `fill-extrusion` layer NOT added. Standard `fill` layer shown. |
| 3D mode, height column set | `fill-extrusion` companion layer added. Standard `fill` layer hidden (or opacity 0) to avoid z-fighting. |
| Below zoom 14 | `fill-extrusion` layer not rendered (minzoom: 14). Standard fill layer always visible regardless of zoom. |

## Animation

### 3D Toggle Pitch Transition

When TerrainControl activates 3D mode, animate the pitch from 0° to 45° using MapLibre's `easeTo`:

```typescript
map.easeTo({ pitch: 45, duration: 300, easing: t => t * (2 - t) });
```

When deactivating, animate pitch back to 0°:

```typescript
map.easeTo({ pitch: 0, bearing: 0, duration: 300, easing: t => t * (2 - t) });
```

Duration: 300ms. Easing: ease-out (quadratic deceleration). Bearing resets to 0 on deactivation.

Source: Claude's Discretion — smooth transition selected over instant snap for spatial orientation continuity.
