4. **Add the height column dropdown JSX** inside the polygon/fill section of the render output, AFTER the existing fill color / stroke controls, BEFORE the section closing tag. Only render when `isPolygon && numericColumns.length > 0`:

```tsx
{isPolygon && numericColumns.length > 0 && (
  <div className="flex items-center justify-between gap-2">
    <span className="text-xs text-muted-foreground">Height column</span>
    <Select
      value={currentHeightCol}
      onValueChange={(val) => {
        const newPaint = { ...layer.paint };
        if (val === '' || val === '__none__') {
          delete newPaint['_height_column'];
        } else {
          newPaint['_height_column'] = val;
        }
        onPaintChange(layer.id, newPaint);
      }}
    >
      <SelectTrigger className="h-8 text-xs w-36">
        <SelectValue placeholder="None" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="__none__">None</SelectItem>
        {numericColumns.map((col) => (
          <SelectItem key={col.name} value={col.name}>
            {col.name}
          </SelectItem>
        ))}
      </SelectContent>
    </Select>
  </div>
)}
```
Note: Use `"__none__"` as the "None" value since shadcn Select does not allow empty string as a value. When `__none__` is selected, delete `_height_column` from paint.
