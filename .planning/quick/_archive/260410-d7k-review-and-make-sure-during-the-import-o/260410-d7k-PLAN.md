**2. `backend/app/ingest/metadata.py` — add `rename_reserved_columns` helper.**

Add this function (placement: right after `ensure_geom_column`, before `clip_to_mercator_bounds`):
```python
async def rename_reserved_columns(
    session: AsyncSession,
    table_name: str,
    *,
    known_source_columns: list[dict] | None = None,
) -> list[dict]:
    """Rename any source column whose name collides with a GeoLens-internal
    PostGIS column (gid, geom, geometry, geom_4326, fid, ogc_fid) to
    `src_<name>`. Runs BEFORE add_4326_column so that ALTER TABLE ADD COLUMN
    geom_4326 cannot collide with a source attribute.

    Only renames columns that were NOT created by the ingest pipeline itself.
    The pipeline guarantees `gid` (via -lco FID=gid), `geom` (via -lco
    GEOMETRY_NAME=geom or ensure_geom_column), and `geom_4326` (added later
    by add_4326_column). Any OTHER column in information_schema with one of
    these names must have come from the source file.

    How we tell them apart:
    - `gid`: if the column exists AND is NOT the PG serial/identity created
      by ogr2ogr's FID=gid. Cross-check by reading
      information_schema.columns for column_default and is_identity; a
      source-origin `gid` will have neither set. (ogr2ogr creates gid as a
      PRIMARY KEY with a nextval default.)
    - `geom`, `geometry`: if the column data_type is NOT `USER-DEFINED` / its
      udt_name is NOT `geometry`. Source-origin `geom` attributes are text,
      numeric, etc., not PostGIS geometry.
    - `geom_4326`: ALWAYS rename on this function's entry path — this helper
      runs BEFORE add_4326_column, so at call time any existing `geom_4326`
      column must have come from the source file.
    - `fid`, `ogc_fid`: always rename (ogr2ogr with -lco FID=gid does NOT
      create these; any such column is source-origin).

    Returns a list of rename records:
      [{"original": "gid", "renamed": "src_gid"}, ...]
    which callers can attach to job.user_metadata['warnings'].
    """
```

Implementation details the executor must follow:

- Use `_validate_table_name(table_name)` at entry.
- Query `information_schema.columns` filtered to `table_schema='data' AND table_name=:t`, selecting `column_name, data_type, udt_name, column_default, is_identity`.
- For each row whose `column_name` is in `RESERVED_COLUMN_NAMES` (import the set from `app.ingest.ogr`), decide whether it is source-origin per the rules above.
- For each source-origin column, compute a safe target name: start with `src_<name>`; if that name already exists in the table (check against the full column list you already fetched), append `_2`, `_3`, ... until unique.
- Execute `ALTER TABLE data.<table_name> RENAME COLUMN "<original>" TO "<target>"` using parameterized-as-best-as-possible SQL. Identifiers cannot be bound, so double-quote them via `'"' + name.replace('"', '""') + '"'`.
- After rename, call `structlog.stdlib.get_logger(__name__).warning("Renamed reserved source column", table=table_name, original=original, renamed=target)`.
- Return the list of renames (empty list if no renames).
- Commit the session at the end.

async def _load_fixture(test_db_session, fixture_name: str, table: str) -> dict:
    """Run the real ogrinfo + ogr2ogr pipeline against the test PostGIS DB.

    Returns a dict with keys:
      - table_name
      - filtered_column_info: output of get_column_info (what catalog sees)
      - raw_column_names: list[str] from information_schema.columns
      - srid
    """
    from app.ingest.metadata import get_column_info
    from app.ingest.ogr import build_pg_conn_str, run_ogr2ogr, run_ogrinfo

    source = str(FIXTURES / fixture_name)
    info = await run_ogrinfo(source)
    await run_ogr2ogr(
        source,
        table,
        build_pg_conn_str(),
        source_srid=info.get("srid"),
        geometry_type=info.get("geometry_type"),
    )
    filtered = await get_column_info(test_db_session, table)
    raw = await test_db_session.execute(
        text(
            "SELECT column_name FROM information_schema.columns "
            "WHERE table_schema='data' AND table_name=:t "
            "ORDER BY ordinal_position"
        ).bindparams(t=table),
    )
    return {
        "table_name": table,
        "filtered_column_info": filtered,
        "raw_column_names": [r[0] for r in raw.all()],
        "srid": info.get("srid"),
    }


async def _drop_table(test_db_session, table: str) -> None:
    await test_db_session.execute(text(f"DROP TABLE IF EXISTS data.{table} CASCADE"))
    await test_db_session.commit()

    """RESEARCH §2.5: get_sample_values returns values for non-ASCII column names."""

    async def test_non_ascii_column_has_sample_values(self, test_db_session):
        from app.ingest.metadata import get_column_info, get_sample_values

        finally:
            await _drop_table(test_db_session, table)
