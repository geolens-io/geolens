    ```ts
    const existing = layersRef.current.find((l) => l.id === layerId);
    if (!existing) return;
    const updated = updater(existing);
    setLocalLayers((prev) => prev.map((l) => (l.id === layerId ? updated : l)));

The pre-check approach uses `layersRef` (the synchronous mirror at line 31–34 maintained for the memoized-callback pattern), is synchronous, and also avoids a no-op state-write when the id misses.
