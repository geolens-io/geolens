<objective>
Add show_in_legend support to the map builder so all visible layers appear in the legend (not just data-driven ones), with a per-layer toggle in the More Actions menu and persistence via the backend.

Purpose: Currently simple single-color layers are invisible in the legend, making the legend incomplete. This makes every layer visible by default and gives users control over which layers appear.
Output: Backend migration + schema changes, shared icon component, legend rendering for simple layers, toggle UX, public viewer filtering.
</objective>
