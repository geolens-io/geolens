<objective>
Add Excel (.xlsx/.xls) file upload and non-spatial ingestion support.

Purpose: Extend the data catalog to accept spreadsheet data, building on the non-spatial CSV pipeline shipped in 260322-hv0. GDAL natively supports Excel via XLSX/XLS drivers (verified in Docker container). JSON ingestion is deferred — GDAL does not support plain JSON.

Output: Users can upload Excel files, preview columns/sample rows, select a sheet for multi-sheet workbooks, and ingest tabular data into PostGIS.
</objective>
