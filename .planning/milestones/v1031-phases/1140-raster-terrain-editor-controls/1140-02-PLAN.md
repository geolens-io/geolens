<objective>
Add EDITOR-DEM-04: a CONTOUR LINES authoring section to `DEMEditorScene` (hillshade + terrain modes) and the companion MapLibre `line` layer that renders client-side contours from the existing raster-dem tiles via `maplibre-contour`.

The control writes builder-private `_contour-*` paint keys (never reaching MapLibre paint directly); a new `contour-sync.ts` module owns the DemSource registration (lazy, keyed by sourceId) and the add/remove/update of the companion line layer.

```
import mlcontour from 'maplibre-contour';
const demSource = new mlcontour.DemSource({ url: demTileUrl, encoding: 'mapbox', maxzoom: 18, worker: true });
demSource.setupMaplibre(map);            // call ONCE per source, AFTER map ready (Pitfall 4)
const url = demSource.contourProtocolUrl({ thresholds: { 9:[500], 11:[100,500], 13:[50,100] }, contourLayer: 'contours', elevationKey: 'ele', levelKey: 'level', overzoom: 1 });
```
