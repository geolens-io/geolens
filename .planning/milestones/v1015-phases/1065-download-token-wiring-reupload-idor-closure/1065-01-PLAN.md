<objective>
Wire the broken-in-production COG download token endpoint and update the frontend to mint-then-open.

Purpose: The existing `create_download_token` helper (auth/service.py:78-111) is unwired. Frontend
`downloadCog()` currently passes the session JWT as `?token=`, which router_export.py:185-201 rejects
because it requires `typ='download'`. This closes a P0 UX regression (raster downloads fail for every user).

- New `POST /api/auth/download-token/{dataset_id}` endpoint in auth router
- `DownloadTokenResponse` Pydantic model in auth schemas

- pytest suite for the endpoint (happy path, no-access 404, anon-public, rate-limit shape)
- Playwright regression pinning the two-request order
- Regenerated OpenAPI snapshot
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → /auth/download-token/{id} | Session JWT in Bearer header; minted download token returned in JSON body |
| browser → /download/cog?token= | Minted download token in URL query param; server validates typ+scope+TTL |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1065-01 | Information Disclosure | download token in URL/logs | mitigate | 120s TTL cap enforced in create_download_token:97; dataset-scope binding via `scope='dataset:{id}'` prevents reuse across datasets; Playwright regression pins that session JWT is not substituted |
| T-1065-02 | Elevation of Privilege | anonymous caller minting token for private dataset | mitigate | check_dataset_access_or_anonymous raises 404 before create_download_token is called; pytest test_download_token_anonymous_private covers this path |
| T-1065-03 | Denial of Service | token farming via repeated POST /auth/download-token | mitigate | 60/min per-IP rate limit via slowapi limiter, same pattern as SEC-S10/S11 in v1014 |
| T-1065-04 | Tampering | session JWT substituted on /download/cog?token= | mitigate | router_export.py:186-192 enforces typ='download'; Playwright test S-IA-P001 asserts 401 on session JWT; no change to router_export.py needed |
| T-1065-SC | Tampering | npm installs | accept | no new npm packages introduced in this plan |
</threat_model>
