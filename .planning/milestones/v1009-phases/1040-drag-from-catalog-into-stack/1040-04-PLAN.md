| Boundary | Description |
|----------|-------------|
| aria-live region content | Strings come from i18n + dataset name; passed via React text content (no innerHTML), so no XSS risk |
| KeyboardSensor focus capture | @dnd-kit's KeyboardSensor takes focus during keyboard drag; risk = trapping focus if drag is not cancelled properly |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-1040-10 | Denial of Service | KeyboardSensor focus trap | mitigate | handleDragCancel (Escape) and handleDragEnd both clear dragActiveId and dragging-active class; verify in test that Escape during keyboard drag restores normal tab order |
