<objective>
Lock the "CSP frame-ancestors NEVER contains `*`" invariant (ROADMAP SC #1, CONTEXT.md locked decision) with a backend defensive pin that survives any future drift in `_validate_origins`. Today the schema accepts whatever passes URL parsing (no explicit wildcard rejection); this plan adds (a) explicit `*` rejection in `_validate_origins` returning a 422 with message "Wildcard origin not allowed" and (b) a second-line-of-defense filter in `_build_frame_ancestors` that drops any stray `*` entry before emitting the header.

Purpose: CSP `frame-ancestors *` is a hard security violation — it disables clickjacking protection entirely. SEC-S08 / Phase 1062-05 builds the per-token directive but does not currently filter `*`. The HARD INVARIANT from ROADMAP says "CSP frame-ancestors NEVER `*`"; this plan turns that invariant into a deterministic test pin.

Output: extended `_validate_origins` schema validator; extended `_build_frame_ancestors` helper with explicit `*` drop; new test file `backend/tests/test_embed_tokens_csp_no_wildcard.py` with 4 tests; existing `test_embed_framing_csp.py` extended with one positive-form pin asserting the literal substring `*` is absent from the shared-map response CSP header.
</objective>

```python
def _normalize_origin(origin: str) -> str:
    normalized = origin.strip().lower().rstrip("/")
    if not normalized.startswith(("http://", "https://")):
        normalized = f"https://{normalized}"
    parsed = urlparse(normalized)
    if not parsed.hostname:
        raise ValueError(f"Invalid origin: {origin}")
    # ... port handling ...

def _validate_origins(v: list[str] | None) -> list[str] | None:
    if v is None:
        return None
    cleaned = []
    for origin in v:
        s = origin.strip()
        if not s:
            continue
        cleaned.append(_normalize_origin(s))
    return cleaned or None
```

From backend/app/modules/catalog/maps/router.py:110-123:
```python
def _build_frame_ancestors(origins: list[str] | None) -> str:
    if not origins:
        return "frame-ancestors 'self'"
    safe: list[str] = []
    for o in origins:
        if "\r" in o or "\n" in o or not o.strip():
            continue
        safe.append(o.strip())
    if not safe:
        return "frame-ancestors 'self'"
    return f"frame-ancestors 'self' {' '.join(safe)}"
```

This is the function that emits the CSP header on GET /maps/shared/{token}. Today it filters CRLF but not `*`. Adding `*` to the dropped-entries set is the defense-in-depth pin.

Existing test_embed_framing_csp.py covers 5 cases — none assert "header does NOT contain '*'". The negative-form pin is the load-bearing addition.
