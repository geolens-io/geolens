# Support

Use the support channel that matches the request.

| Need | Where |
|------|-------|
| Questions, setup help, and usage discussion | [GitHub Discussions](https://github.com/geolens-io/geolens/discussions) |
| Feature ideas and product feedback | [GitHub Discussions: Ideas](https://github.com/geolens-io/geolens/discussions/categories/ideas) |
| Reproducible bugs | [GitHub Issues](https://github.com/geolens-io/geolens/issues) |
| Security vulnerabilities | Email **security@getgeolens.com** and follow the [security policy](.github/SECURITY.md) |
| Enterprise support | Email **enterprise@getgeolens.com** |

Full product documentation lives at [docs.getgeolens.com](https://docs.getgeolens.com).

Please search existing issues and discussions before opening a new thread. Community support is best-effort.
